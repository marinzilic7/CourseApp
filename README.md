## Applikacija

Jednostavana aplikacija za kolegij "Projektiranje sustava e-učenja"

## About App

Aplikacija za tečajeve. Korsnik mora da bude registriran i logiran kako bi dodavao, update i brisao kurseve. 
## Tehnologije 

- **Laravel**
- **Vue**
- **MySQL** 
- **Bootstrap** 

## Funkcionalnosti

- **Register i Login** 
    - **Poruke o greškama**
- **Dodavanje kursa**
- **Update kursa**
- **Brisanje kursa**
- **Odjava korisnika**
